# Athena Plugin - Cayo Perico

A cayo perico plugin for the Athena Framework compatible with `4.0.0` of the [Athena Framework](https://athenaframework.com/).

Enables visiting the cayo perico island.

## Installation

1. Open a command prompt in your main Athena Directory.
2. Navigate to the plugins folder.

```ts
cd src/core/plugins
```

3. Copy the command below.

**SSH**

```
git clone git@github.com:Athena-Roleplay-Framework/athena-plugin-cayo-perico.git
```

**HTTPS**
```
git clone https://github.com/Athena-Roleplay-Framework/athena-plugin-cayo-perico
```

4. Start the Server
